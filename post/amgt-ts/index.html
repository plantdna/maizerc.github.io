AMGT-TS User manual

We provided two methods to use our tool. One is the online version, the other is the terminal version. So we offer two guides for each method. The guide for the online version can be found here, and the guide for the terminal version can be found here.

Note: The tool needs two type of input: one is the FASTQ file, the other is the information files of the markers. Information about FASTQ file type can be found on illumina, or on wikipedia.

Information files of the markers are explained as following:

  1. Sequence file about the markers, example on github as:

This is in fact a FASTA file format(https://en.wikipedia.org/wiki/FASTA_format), each sequence correspond a marker. The SSR region in the sequence marks as lower case bases, and blanks are marker as upper case bases.

  1. Stats information about the markers, example as https://github.com/plantdna/amgt-ts/blob/master/ref/sites.motif.info.stats:
#ID SSR.No Motif_len Motif Repeat_times Start End Repeat_len Seq_len
s258878 1 3 GCT 5 301 315 15 615
s282049 1 3 TGG 5 301 315 15 615

It has 9 columns, meaning as:

  • 2.1 #ID The marker id, same as the marker’s name in the Sequence file of the markers.

  • 2.2 SSR.No The motif index of the marker. We support multi-motif in one SSR region. But for the common use case, the value of this column is always 1.

  • 2.3 Motif_len The bases count of the motif.

  • 2.4 Motif Thre repeat unit of the marker.

  • 2.5 Repeat_times The repeat times of the motif in this marker.

  • 2.6 Start The start base position of the SSR region in the sequence of the marker. From the first base to position Start -1, is the left flank of the marker.

  • 2.7 End The end base position of the SSR region in the sequence of the marker. From End +1 to the last bases of the marker, if the right flank of the marker.

  • 2.8 Repeat_len The result of column #3 (Motif_len) times column #5 (Repeat_times)

  • 2.9 Seq_len The length of the sequence, is the bases count of the marker in the Sequence file about the marker.
